# GitHub 今日趋势：2026-04-25 热门开源项目精选

## 1. [free-claude-code](https://github.com/Alishahryar1/free-claude-code)
- **主要语言**: Python
- **今日新增 Stars**: 2638
- **简介**: 该项目允许开发者在终端、VSCode 插件或通过类似 OpenClaw 的 Discord 机器人免费使用 Claude Code。它为追求高效编码体验的开发者提供了一个无需订阅即可体验先进 AI 代码辅助工具的途径，因其极高的实用性和便捷性在今日霸榜。

- **今日新增 Stars**: 2985

## 3. [osv-scanner](https://github.com/google/osv-scanner)

- **今日新增 Stars**: 141
- **简介**: Google 开发的漏洞扫描器，利用 osv.dev 提供的海量数据来识别开源项目的安全风险。对于重视供应链安全的企业和个人开发者来说，这是一个不可或缺的自动化安全审查工具。
