# GitHub 今日趋势：2026-03-31 热门开源项目精选

三月的最后一天，GitHub 社区迎来了一波“终端 AI 助手”的热潮。开发者们正致力于将大模型的能力无缝集成到日常命令行工具中。

## 1. [anthropics/claude-code](https://github.com/anthropics/claude-code)
- **主要语言**: Shell
- **今日 Stars**: 10087
- **简介**: Anthropic 官方推出的终端 AI 编程工具。它能直接读取你的代码库，通过自然语言指令完成常规任务、解释代码逻辑并处理 Git 工作流。今日 Stars 数过万，足以说明其震撼程度。

## 2. [luongnv89/claude-howto](https://github.com/luongnv89/claude-howto)
- **主要语言**: Python
- **今日 Stars**: 3336
- **简介**: 一份视觉化的、由示例驱动的 `Claude Code` 使用指南。从基础概念到高级 Agent 技巧，提供了大量可直接复用的模板，是目前学习官方工具的最佳非官方手册。

## 3. [openai/codex](https://github.com/openai/codex)
- **主要语言**: Rust
- **今日 Stars**: 2345
- **简介**: OpenAI 同样推出了运行在终端的轻量级编程助手。与 Claude Code 形成直接竞争，采用了 Rust 编写以追求极致的运行速度和资源效率。

## 4. [microsoft/VibeVoice](https://github.com/microsoft/VibeVoice)
- **主要语言**: Python
- **今日 Stars**: 1704
- **简介**: 微软开源的前沿语音 AI。它代表了目前开源界在语音合成与识别方面的最高水平，能够提供极其自然且具有情感表现力的语音交互体验。

## 5. [google-research/timesfm](https://github.com/google-research/timesfm)
- **主要语言**: Python
- **今日 Stars**: 358
- **简介**: Google Research 发布的预训练时间序列基础模型。它专门用于时间序列预测，为金融预测、气象分析等领域提供了强大的 AI 底座。
