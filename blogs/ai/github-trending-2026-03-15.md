## 1. [OpenViking](https://github.com/volcengine/OpenViking)
- **主要语言**: Python
- **今日 Stars**: 1610 stars today

**项目介绍**: OpenViking 是由火山引擎开源的、专门为 AI Agent（如 OpenClaw）设计的上下文数据库。它借鉴了文件系统的范式来统一管理 Agent 所需的记忆、资源和技能。

**技术亮点**: 通过层级化的上下文交付机制，实现了 Agent 的自我进化能力。其文件系统化的管理方式让 Agent 的“思维链”和“知识库”更具条理且易于扩展。

**使用场景**: 适合需要构建复杂、长生命周期 Agent 的开发者，解决 Agent 在处理海量任务时上下文管理混乱的问题。

## 2. [agency-agents](https://github.com/msitarzewski/agency-agents)
- **主要语言**: Shell
- **今日 Stars**: 4280 stars today

**项目介绍**: 这是一套完整的“AI 机构”解决方案。它提供了一系列高度专业化的 Agent，包括前端专家、Reddit 社区运营专家、甚至还有专门负责“现实检查”的审计 Agent。

**使用场景**: 适合初创企业或独立开发者，通过部署这套 Agent 矩阵来快速搭建从开发到营销的全流程自动化流水线。

- **今日 Stars**: 2069 stars today
