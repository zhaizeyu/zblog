## 1. [abhigyanpatwari / GitNexus](https://github.com/abhigyanpatwari/GitNexus)
- **主要语言**: TypeScript
- **今日 Stars**: +1,116
- **项目简介**: GitNexus 是一个运行在浏览器中的客户端知识图谱生成器。它允许开发者直接拖入 GitHub 仓库或 ZIP 文件，即可生成交互式的代码知识图谱。
- **技术亮点**: 集成了内置的 Graph RAG（检索增强生成）智能体，能够深入分析代码结构。
- **使用场景**: 快速上手新项目、进行深度的代码架构分析或探索复杂的开源库。

## 2. [langchain-ai / deepagents](https://github.com/langchain-ai/deepagents)
- **主要语言**: Python
- **今日 Stars**: +1,415
- **项目简介**: 由 LangChain 官方推出的深层智能体框架。它基于 LangGraph 构建，提供了一个强大的 Agent 运行环境。
- **技术亮点**: 包含精密的规划工具、文件系统后端，并支持生成子智能体（subagents）来处理复杂任务。
- **使用场景**: 构建能够自主处理多步骤、复杂逻辑的 AI 应用，特别适合需要高度自主决策的场景。

## 3. [jarrodwatts / claude-hud](https://github.com/jarrodwatts/claude-hud)
- **主要语言**: JavaScript
- **今日 Stars**: +466
- **项目简介**: 这是一个专为 Claude Code 打造的实时状态仪表盘（HUD）插件。
- **技术亮点**: 能直观展示上下文使用量、活跃工具、正在运行的智能体以及待办事项进度。
- **使用场景**: 开发者在使用 AI 编程助手时，可以通过该插件实时监控 AI 的工作状态和资源消耗，提高协作效率。

- **使用场景**: 构建和测试边缘计算应用，或者在本地模拟 Cloudflare 的生产环境。
