## 1. [browser](https://github.com/lightpanda-io/browser)
- **主要语言**: Zig
- **今日 Stars**: 1335 stars today

Lightpanda 是一款专为 AI 和自动化设计的无头浏览器。它使用 Zig 语言编写，追求极致的性能和低资源占用。与传统的无头浏览器相比，它更轻量，能够更高效地集成到各种 AI 流程中。

**使用场景**: 自动化网页抓取、AI 代理网页交互、大规模自动化测试。

- **今日 Stars**: 205 stars today

**使用场景**: 应急通信、离线知识库、偏远地区计算需求、数字生存备份。

## 3. [OpenViking](https://github.com/volcengine/OpenViking)
- **主要语言**: Python
- **今日 Stars**: 1870 stars today

OpenViking 是由火山引擎（Volcengine）开源的专门为 AI Agent 打造的上下文数据库。它通过文件系统范式统一管理 Agent 所需的记忆、资源和技能等上下文，支持层次化的上下文交付和自我演进。

**使用场景**: AI 智能体长短期记忆管理、复杂任务上下文同步、Agent 技能自进化。
