# 博客

这里集中收录站点中的博客内容，当前采用「按分类组织、按年份归档」的方式展示：

- `ai`：每日 GitHub 趋势与 AI 方向观察。
- `deployment-guides`：各类项目部署、运维与实践手册。

> 侧边栏会在 VitePress 启动与构建时自动扫描 `blogs/` 目录并生成，因此新增文章后无需手动改配置。

## 写作建议

1. 将新文章放到合适的分类目录中，例如 `blogs/ai/`。
2. 在文章顶部使用一级标题，脚本会优先读取该标题作为侧边栏名称。
3. 若需要归档分组，可继续使用像 `blogs/2026/` 这样的年份目录结构，脚本会自动识别年份。
