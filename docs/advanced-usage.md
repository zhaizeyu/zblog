# 进阶用法

## 自动侧边栏的工作方式

站点会在加载 `.vitepress/config.mts` 时读取 `scripts/content.mjs`：

- 自动遍历 `blogs/` 中的 Markdown 文件。
- 优先读取文章的一级标题作为导航名称。
- 根据一级目录生成分类分组。
- 识别四位年份目录并自动生成归档分组。
- 自动读取 `docs/` 文档，生成独立的文档侧边栏。

## 推荐写法

- 博客文章尽量只保留一个一级标题。
- 分类目录命名保持稳定，例如 `ai`、`deployment-guides`。
- 如需更丰富的展示，可以继续给文章补充 frontmatter，后续再扩展脚本读取字段。
